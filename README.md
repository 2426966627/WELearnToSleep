# WELearnToSleep

## 简介
* 输入账号密码一键完成课程
* 可以自行选择课程
* 正确率显示为100%
* 已完成的章节会自动跳过

**PS:本软件只完成课程，不会做题，如需做题请尝试显示答案的脚本食用**
